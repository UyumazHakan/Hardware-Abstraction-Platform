## How to run the application
```bash
# Get the latest snapshot
git clone --depth=1 https://gitlab.lrz.de/IoT-Practicum-Group/sensors.git project

# Change directory
cd project/web_server

# Install NPM dependencies
npm install

# Then simply start app with
node app.js 
or 
nodemon app.js (for live-reload)

# browse to http://localhost:3000
```
